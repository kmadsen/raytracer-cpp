## About

Setting up an old ray-tracer project to work that I built in college.

Project 4 introducing lense blur and area lights.

## Building

``` bash
$ cd project04
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## Running

``` bash
$ cd project04
$ ./build/bin/ray-tracing prog04.scn 
$ eog image.ppm
```
