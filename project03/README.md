## About

Setting up an old ray-tracer project to work that I built in college.

Project 3 included introduces a bounding volume hierarchy.

## Building

``` bash
$ cd project03
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## Running

``` bash
$ cd project03
$ ./build/bin/ray-tracing prog03.scn 
$ eog image.ppm
```
